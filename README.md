# EduManage – Learning Management System

**EduManage** is a console-based **Learning Management System (LMS)** developed in Java. It allows users to manage students, courses, and enrollments efficiently using object-oriented programming concepts.

## Features

- Add and manage students.
- Add and manage courses.
- Enroll students in courses.
- View detailed information about students and their enrolled courses.
- View detailed information about courses and their enrolled students.
- Simple and interactive console-based interface.
- Modular design using object-oriented principles.

## Technologies & Skills Used

- **Java** – Core programming language.
- **Object-Oriented Programming (OOP)** – Classes, Objects, Encapsulation, and Methods.
- **Data Structures** – `ArrayList` for managing collections of students and courses.
- **Console I/O** – `Scanner` for user input.
- **Control Flow** – Loops, conditionals, and switch-case for menu-based navigation.
- **Problem-Solving & Logical Thinking** – Designing enrollment and management logic.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/EduManage-LMS.git
   ```
2. Navigate to the project folder:
   ```
   cd EduManage-LMS
   ```
3. Compile all Java files:
   ```
   javac *.java
   ```
4. Run the application:
   ```
   java LMS
   ```
   
## Usage
- Open the application in the terminal.

- Choose options from the menu:
- Add a student.
- Add a course.
- Enroll a student in a course.
- View students and courses.
- Follow on-screen instructions to perform actions.
- Repeat actions as needed until you choose to exit.
